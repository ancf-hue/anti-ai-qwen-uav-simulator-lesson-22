## 📋 О проекте

**Anti-AI Qwen UAV Simulator** — это интерактивный 3D-тренажёр оператора беспилотных летательных аппаратов, разработанный в рамках **Урока 22** обучающей программы Qwen. 

Игрок выступает в роли **оператора Охотник**, задача которого — обнаружить, догнать и уничтожить все вражеские дроны в отведённое время. Тренажёр имитирует реальное пилотирование БПЛА с детальной физикой полёта, системой повреждений и динамическим ИИ противников.

> ⚠️ **ВНИМАНИЕ:** Проект создан исключительно в образовательных целях.

## ✨ Особенности

### 🎮 Геймплей
- 🎯 **3 уровня сложности** с разным количеством и скоростью врагов
- 🚁 **Детализированная 3D-модель дрона** с вращающимися пропеллерами
- 🌲 **Динамическое окружение** — деревья как препятствия
- 💥 **Система повреждений** — столкновения с деревьями снижают здоровье
- 🏆 **Система очков и рекордов** с сохранением в `localStorage`
- ⏱️ **Таймер миссии** с предупреждениями о времени

### 🎨 Визуал
- 🌅 **Кинематографичное освещение** с тенями (PCFSoftShadowMap)
- 🎭 **ACES Filmic тонмаппинг** для реалистичной картинки
- 💫 **Анимированные UI-элементы** с градиентами и свечением
- 🌫️ **Экспоненциальный туман** для атмосферы
- 🎨 **Неоновые пропеллеры** с прозрачностью

### 🔊 Аудио
- 🎵 **Фоновая музыка** (опционально)
- 🔔 **Процедурно генерируемые звуки** через Web Audio API:
  - Взрывы врагов
  - Столкновения с препятствиями
  - Победная мелодия
  - Предупреждения

### 🧠 Технологии
- ⚡ **Three.js r128** — 3D-рендеринг
- 🎭 **Web Audio API** — синтез звуков
- 💾 **localStorage** — сохранение рекордов
- 📱 **Адаптивный дизайн** — работает на мобильных устройствах

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `W` / `↑` | Полёт вперёд |
| `S` / `↓` | Полёт назад |
| `A` / `←` | Стрейф влево |
| `D` / `→` | Стрейф вправо |
| `Q` / `Shift` | Набор высоты |
| `E` / `Ctrl` | Снижение |
| `ЛКМ + движение мыши` | Обзор камеры 360° |
